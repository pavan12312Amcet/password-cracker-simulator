# 🔐 Password Cracker Simulator

A Python-based project that demonstrates how weak passwords can be cracked using dictionary attacks and SHA-256 hash techniques.

---

## 🚀 Features

* Dictionary Attack
* Hash Cracking (SHA-256)
* Password Strength Checker

---

## 🛠️ Tech Stack

* Python
* hashlib

---

## ▶️ How to Run

python main.py

---

## 🧠 How It Works

* Takes a password or hash as input
* Uses a wordlist to guess passwords
* Converts each guess into hash
* Matches with target hash
